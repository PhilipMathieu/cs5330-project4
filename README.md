IDE: VS Code

# How to Run

To run the calibration program:
```
make calibrate
bin/calibrate.out
```

To run the augmented reality program:
```
make augment
bin/augment.out
```

To run the Harris Corners program:
```
make harris
bin/harris.out
```

To run the `python` implementation of the augmented reality program:
```
make python
```
